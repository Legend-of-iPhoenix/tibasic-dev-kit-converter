_iPhoenix_'s Code Converter/Optimizer Thing

Instructions:

Use TI-Connect CE to grab your programs from your calculator.

Upload your code to SourceCoder 3, and copy the result. This will convert your code into a usable format.

Paste your code into the first textbox, and press submit.

Copy the code in the second textbox, and paste it into SourceCoder 3. Be sure to delete the old code.

If everything went well, you can press the export project icon, and it will download the .8xp file to your computer.

Transfer the downloaded file to your calculator with TI-Connect CE, and you are all set!
